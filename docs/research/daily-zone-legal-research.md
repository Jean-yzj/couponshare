# CouponShare「當日專區」台灣法律風險研究報告

> 研究對象：讓使用者分享/轉讓「當天取得、當天過期」的券（典型例：停車場消費贈送的停車折抵券）。
> 研究日期：2026-07-16。以中華民國現行法規為準。
> 研究方法：全國法規資料庫（law.moj.gov.tw）逐字比對條文為主，輔以政府機關公告與律師事務所專業文章。
> **重要免責：本報告為法律資訊整理，非正式法律意見。上線前，尤其是決定是否開放「加價出售」與高風險類別時，建議諮詢執業律師。**

---

## 摘要（TL;DR）

- **停車折抵券 / 商家消費贈送的優惠券，不在任何「禁止黃牛加價轉售」的特別法範圍內。** 現行反黃牛特別法（文創法、運動產業條例、鐵路法）只管各自明列的票券類型（藝文表演票、運動賽事票、車票），不及於折價券/兌換券。
- **這類「折價券／抵用券」在法律上與「預付型商品禮券」是不同的東西。** 經濟部禮券定型化契約規範**明文把「無償發行之抵用券、折扣（價）券」排除在「禮券」定義之外**，停車折抵券正落在被排除的一方。
- **平台最大的風險不是「券本身能不能轉」，而是使用者上架「已使用/偽造/無效」的券造成他人受損時的平台責任。** 台灣的 ISP 避風港（著作權法第六章之一）**只涵蓋著作權/製版權侵害，不涵蓋詐欺或一般民事侵權**，所以沒有現成的法定免責傘可用。平台要靠「單純中介地位 + 善良管理 + 通報下架」在個案中爭取無過失，而非援引特定免責條款。
- **應主動排除的高風險類別：菸品（含電子煙/加熱菸，網售全面違法）、酒類（現行禁止網售）、藥品與醫療器材（非藥商網售違法）、彩券（禁止轉讓/受讓）。** 醫美療程券、金融商品、成人商品風險較高，建議一併排除或嚴管。

## 問題 1：反黃牛/加價轉售法規的範圍，涵蓋停車折抵券嗎？

### 已查證的事實

台灣目前針對「票券加價轉售（黃牛）」有三部特別法，各自的規範對象都**限定於特定票券類型**，逐字比對如下：

**(1)《文化創意產業發展法》第10條之1**（112年6月2日施行）— 全國法規資料庫 pcode H0170075
逐字條文：「政府應致力於保障民眾近用文化創意活動之權益，確保**藝文表演票券**正常流通。將**藝文表演票券**以超過票面金額或定價販售者，按票券張數，由主管機關處票面金額或定價之十倍至五十倍罰鍰。以虛偽資料或其他不正方式，利用電腦或其他相關設備購買**藝文表演票券**，取得訂票或取票憑證者，處三年以下有期徒刑，或科或併科新臺幣三百萬元以下罰金……」
→ 規範對象**僅限「藝文表演票券」**（現場音樂、戲劇、舞蹈等表演；演唱會在內）。

**(2)《運動產業發展條例》第24條之1**（pcode H0120056）
逐字條文（節錄）：「政府應致力於保障民眾觀賞**運動賽事或活動**之權益……將公開販售之**運動賽事或活動票券**，以超過票面金額……販售者，按票券張數，由主管機關處……十倍至五十倍罰鍰。將全面或部分免費入場之**運動賽事或活動免費票券**定價販售者，按票券張數，由主管機關處每張票券新臺幣一萬八千元以下罰鍰。以虛偽資料或其他不正方式……購買運動賽事或活動票券……處三年以下有期徒刑，或科或併科新臺幣三百萬元以下罰金……」
→ 規範對象**僅限「運動賽事或活動票券」**。

**(3)《鐵路法》第65條**（pcode K0030001）
現行逐字條文：「**車票**加價出售或換取不正利益圖利者，按車票張數，處其運價之十倍至五十倍罰鍰。加價出售**取票憑證**圖利者，亦同。以不正方法將虛偽資料或不正指令輸入電腦或其相關設備而購買**車票**、取得訂票或取票憑證者，處五年以下有期徒刑或科或併科新臺幣三百萬元以下罰金。」
→ 規範對象**僅限「車票」及取票憑證**（台鐵、高鐵等）。
> 註：不同來源就倍數有「五倍至三十倍」與「十倍至五十倍」兩種說法，疑為修法沿革所致；全國法規資料庫現行版本為「十倍至五十倍」。實際裁罰以最新條文為準。

### 免費轉讓 vs 加價轉售的差別待遇（已查證）

- **文創法與鐵路法**：條文只處罰「以超過票面金額/加價出售」的行為，**未處罰免費/原價轉讓**。
- **運動產業條例**：較嚴，連「將免費入場的免費票券『定價販售』」都罰（每張1.8萬元以下）——但仍是針對「販售（收費）」，單純無償贈送不在內。
- 律師見解（鴻安法律等）：票券轉讓法律不會一律當黃牛，原價轉讓、臨時轉讓、贈送通常不違法，關鍵在有無「營利性質」與「大量收購意圖」。

### 結論
**停車折抵券／商家消費贈送的優惠券，不落在上述任何一部禁止加價轉售的特別法範圍內**（三法皆為列舉式，只管藝文表演票、運動賽事票、車票）。免費轉讓與加價轉售在法律上確有差別：現行反黃牛特別法主要針對「加價販售」；但這個差別**只在那三類票券適用**，對停車折抵券兩者原則上都不觸犯特別法。

**信心：高**（三部條文均逐字比對全國法規資料庫）。
**關鍵法源**：文創法§10-1、運動產業發展條例§24-1、鐵路法§65。

## 問題 2：「禮券」定型化契約規範，適用折價券/停車折抵券嗎？

### 已查證的事實

**經濟部「零售業等商品（服務）禮券定型化契約應記載及不得記載事項」** 的名詞定義（引自經濟部主管法規共用系統 GL001186）：

- 禮券定義：「商品（服務）禮券，指發行人發行**一定金額**之憑證、磁條卡、晶片卡或其他類似性質之證券，由持有人以提示、交付或其他方法，向發行人或其指定之人請求交付或提供**等同於上開證券所載金額**之商品或服務。」
- **明文排除**：「商品（服務）禮券**不包括發行人無償發行之抵用券、折扣（價）券**，及電子票證發行管理條例所稱之電子票證。」

→ 也就是說，「禮券」規範管的是**預付型、載有一定金額、消費者付了對價**的憑證（如百貨禮券、超商禮券），核心目的在保護預付款（履約保證機制、不得記載使用期限等）。

### 折價券與預付型禮券在法規上是不同的東西（已查證）

- **停車折抵券／消費贈送的抵用券**＝商家「無償發行」、用來折抵或兌換的憑證，**正好落在被排除的「無償發行之抵用券、折扣（價）券」一方**，**不適用**禮券定型化契約規範。
- 消費者未就該券支付對價、券也非「一定金額之預付」，因此禮券規範的履約保證、不得限使用期限等保護**對停車折抵券不適用**（這也是為何停車券可以「當天過期」——禮券「不得記載使用期限」的限制對它無效）。

### 結論
**折價券/優惠券/停車折抵券 與 預付型商品禮券在法規上是不同的東西。** 禮券定型化契約規範明文把「無償發行之抵用券、折扣（價）券」排除，停車折抵券屬被排除者，**不受該規範拘束**，也不享有其消費者保護。

**信心：高**（排除文字逐字引自經濟部法規系統）。
**關鍵法源**：經濟部「零售業等商品（服務）禮券定型化契約應記載及不得記載事項」名詞定義段（GL001186）；另有「商品（服務）禮券定型化契約應記載及不得記載事項」（109/4/10公告、110/1/1生效）採相同排除。
> 提醒：禮券相關規範近年有整併/預告修正，上線前宜再確認現行版本，但「排除無償抵用券/折扣券」的核心立場歷來一致。

## 問題 3：平台對「使用者張貼/轉讓的券」的法律責任

### 3a. 上架已使用/偽造/無效的券導致他人受損 — 平台責任與免責

**已查證的事實：**

- **台灣沒有「涵蓋所有網路侵權」的通用平台免責法。** 現行唯一的 ISP 責任限制是**《著作權法》第六章之一「網路服務提供者民事免責事由」**（98年/2009年增訂）。經逐字查證與 Winkler Partners、經濟部智慧局說明確認：此避風港**僅適用於使用者「侵害著作權或製版權」的情形**，**不及於詐欺、一般民事侵權、商品瑕疵**等。智慧局資料亦明列：網路常見違法（誹謗、賭博、妨害電腦使用、違反消保法/公平法等）**均不在此避風港適用範圍**。
  → 即：平台不能援引任何「通報下架就免責」的現成法條，來對「使用者上架假券/廢券詐騙」直接免責。要主張免責需符合的著作權法第90條之4要件（告知保護措施、三振條款、公告聯絡窗口等）也只對著作權案件有意義。

- **民事（侵權）**：依《民法》第184條，須平台自身有故意或過失（例如明知有大量假券仍放任）才構成侵權/共同侵權。**單純提供中介平台、對個別違規無「明知或可得而知」，通常不成立侵權**。但《消費者保護法》上，若平台被認定為「企業經營者」或就廣告內容「明知或可得而知與事實不符」，可能與賣方負連帶責任（消保法第22、23條相關實務）——這點在 C2C 平台是否為「企業經營者」有爭議，屬個案認定。

- **刑事（詐欺幫助）**：刑法詐欺罪須行為人有詐欺故意。平台若僅提供工具、對個別詐騙無認識，**原則上不構成詐欺幫助犯**（幫助犯需對正犯犯罪有認識）。風險升高的情境：平台明知某類券高比例造假仍不處置、或從詐騙中抽成獲利而放任。

**推測（未有直接判例佐證，標示為推測）：**
- 就「當日券」這種**低單價、即時失效、且多半免費轉讓**的標的，實務上單一交易受害金額小，被認定平台需負連帶責任或幫助犯的門檻，**推測**會比高單價商品低很多；但「可驗證性低（券可能已被用掉、真偽難辨）」反而是平台應主動建立防詐機制的理由。這是風險控制判斷，非既有法條明文。

**降低責任的合理做法（實務建議，非法條強制）：**
1. 明確定位為「中介平台」，非賣方/發行人；使用者條款載明券之真偽、有效性由上架者負責。
2. 建立**通知—下架—停權**機制與檢舉入口（比照著作權避風港精神，即使法律不強制，有助於證明無過失）。
3. 保留上架者可識別資訊（利於追究實際加害人、也利於平台切割責任）。
4. 對「已使用/過期」給明顯風險揭露；即時券建議即時失效驗證或限制。

### 3b. 券上載明「不得轉讓/限本人使用」時，使用者轉讓 + 平台是否「誘使/協助違約」

**已查證的事實：**

- **「贈品不得轉售」「限本人使用」的條款，在民事上屬「契約自由/私法自治」，原則上合法有效**（法律百科、著作權筆記等專業文章一致見解）。使用者轉讓即**違反其與發券商家間的契約**。
- **契約相對性（privity）**：此違約責任存在於「使用者 ↔ 發券商家」之間。商家可對違約的使用者主張違約（如作廢券、拒絕兌現）。**平台並非該契約當事人，不直接負違約責任。**
- 商標權角度（進階、屬推測性風險）：有見解認為標明「不得轉售、僅供受贈人使用」之贈品，因非「由商標權人同意於市場交易流通」之商品，轉售**可能**觸及商標權耗盡原則的例外——但此見解適用於「實體商品贈品」的轉售，套用到「折抵券」是否成立並無直接判例，列為推測。

**平台是否構成「誘使/協助違約」（第三人侵害債權）？**
- 台灣民法對「第三人故意引誘他人違約」可能循**民法第184條第1項後段（故意以背於善良風俗之方法加損害於他人）**構成侵權，但門檻高：需平台**明知**該券有不得轉讓條款、且**積極誘使**違約。
- **推測（無直接判例）**：單純提供通用轉讓平台、未針對特定商家的禁轉券做積極促銷，**推測**難認定為「誘使違約」；違約主體與責任仍在轉讓的使用者身上。但若平台明知某商家券明文禁轉，仍以行銷話術鼓勵使用者上架該類券，風險上升。

### 結論
- **免責傘**：台灣無涵蓋詐欺/一般侵權的通用平台免責法；著作權避風港不適用。平台靠「中介定位＋善良管理＋通報下架＋無明知過失」在個案爭取不負責，而非援引免責條款。**信心：高**（ISP 避風港範圍已逐字查證）。
- **禁轉券**：違約主體是轉讓的使用者（契約相對性），平台原則上不負違約責任；「誘使違約」門檻高、需明知並積極促成。**信心：中**（原則清楚，但個案認定與商標權延伸屬灰區，缺直接判例）。
**關鍵法源**：著作權法第六章之一（§90之4以下）；民法§184；消保法§22、§23（廣告連帶責任）；刑法§339（詐欺）與幫助犯法理。

## 問題 4：台灣明確不能公開轉讓 / 有廣告或交易限制的高風險券類別

以下為已查證、平台**建議排除或嚴管**的類別（附法源與罰則）：

| 類別 | 現行限制（已查證） | 法源 / 罰則 |
|---|---|---|
| **菸品／電子煙／加熱菸** | 電子煙、加熱菸等類菸品**全面禁止**製造、輸入、販賣（含網路、面交）、供應、展示、廣告。加熱菸須通過健康風險評估審查始得製售。 | 菸害防制法§15等（112/3/22施行）。販賣/展示處20萬–100萬；廣告/刊載處40萬–200萬（他人20萬–100萬）；製造輸入最高5,000萬。**信心：高** |
| **酒類** | 酒類**不得**以郵購、電子購物等無法辨識購買者年齡之方式販售——即**現行禁止網路販售**。酒廣告須標警語、不得鼓勵飲酒/以兒少為對象等。 | 菸酒管理法（網售違反處1萬–5萬並按次罰；違法酒廣告處3萬–50萬）。**信心：高** |
| **藥品／醫療器材** | 非藥商不得於網路販售藥品/醫療器材；僅具實體店面之販賣業藥商可網售部分第一、二等級醫材。藥物廣告須經核准。 | 藥事法§27、§65、§66、§91、§92、§95。非藥商販藥處3萬–200萬；違法藥品廣告處20萬–500萬。**信心：高** |
| **彩券（公益彩券/運動彩券）** | 發行/受託機構員工**不得購買、受讓或兌領**運動彩券；不得銷售或支付獎金予未成年人。中獎彩券須本人憑身分證明具領。彩券銷售為特許，非經銷商不得販售。 | 公益彩券發行條例、運動彩券發行條例及管理辦法。**信心：高**（禁轉讓/受讓限於機構員工係明文；一般人轉讓中獎彩券的通用禁令未見明文，兌獎採記名具領實質限制轉讓）。 |
| **醫美療程／瘦身美容課程券** | 屬「瘦身美容/美容定型化契約」規範對象（有退費、不得記載事項規範）。療程多具「限本人、需醫療評估」性質，第三人受讓後未必能施作，糾紛與人身安全風險高。 | 衛福部瘦身美容/美容定型化契約應記載及不得記載事項。**信心：中**（未見「禁止轉讓」明文，但實務上限本人、風險高，建議排除）。 |
| **金融商品** | 證券、保險、基金等受金管會高度監理，銷售須特許資格；「券/兌換權」若涉投資性質恐踩證交法/銀行法。 | 證交法、銀行法、保險法等（未逐條查證）。**信心：中—低**（方向明確，個案需個別確認）。 |
| **成人／情趣商品** | 非全面禁止，但受刑法散布猥褻物品罪（刑法§235）、兒少性剝削防制條例、廣告妨害善良風俗等規範，網路展示/廣告易觸法。 | 刑法§235；兒少性剝削防制條例。**信心：中**（未逐條查證，屬方向性建議）。 |

### 平台排除建議（風控結論）
**強烈建議直接排除（有明確網售/交易禁令）：** 菸品（含電子煙、加熱菸）、酒類、藥品與醫療器材、彩券。
**建議排除或嚴管（限本人/高糾紛/監理特許）：** 醫美療程券、金融商品、成人商品。
另建議一併排除：**須實名/限本人使用的證件類、有價金錢等值憑證（可能觸電子票證/儲值監理）、以及任何載明「禁止轉讓」的高單價券**（雖違約主體是使用者，但集中出現易升高平台責任觀感）。
